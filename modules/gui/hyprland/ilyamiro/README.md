# Ilyamiro Hyprland profile

This directory contains the active local Hyprland session files adapted from:

- `config/sessions/hyprland/hyprland.conf`
- `config/sessions/hyprland/config/*.conf`

The larger Quickshell script tree and templates remain sourced from the pinned
`ilyamiro-dots` flake input because that tree is large and tightly coupled.
The local copies here are used as the active compositor layer so host-specific
values can be adapted without modifying the upstream source.
